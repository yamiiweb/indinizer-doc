- Website: https://doc.indinizer.app
- Built with: Laravel 8

- Implements Simple-Sidebar Template from startbootstrap.com

## Installation
```bash
git clone https://github.com/yamiiweb/indinizer-doc
composer install
npm install
cp .env.example .env
php artisan key:generate
```
